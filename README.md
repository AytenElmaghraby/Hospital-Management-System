# 🏥 Hospital Management System

A Java application that demonstrates Object-Oriented Programming principles through a hospital management system.

---

## 📖 Overview

The system manages hospital operations including patients, doctors, appointments, medical records, and room reservations.

---

## ✨ Features

- Login System
- Patient Management
- Doctor Management
- Medical Records
- Room Reservation
- Search
- Sorting
- CRUD Operations

---

## 💡 OOP Concepts

- Inheritance
- Polymorphism
- Abstraction
- Interfaces
- Association
- Aggregation
- Composition
- Overloading
- Overriding

---

## 🛠 Technologies

- Java
- OOP

---


## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/AytenElmaghraby/Hospital-Management-System.git
```

Open the project using NetBeans.

Run the application.
